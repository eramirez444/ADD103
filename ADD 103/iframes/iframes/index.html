Exploring iFrames

Two examples below: an embedded website (OpenStreetMap) and an embedded YouTube video. Both include accessible titles and responsive styling.

Embedded Webpage (OpenStreetMap)

This iFrame displays an external webpage designed for embedding: OpenStreetMap's shareable map.

If you see a “refused to connect” message here, the site is blocking iframes via X-Frame-Options or Content-Security-Policy. Try another public page that allows embedding.

Embedded Video (YouTube)

This iFrame embeds an educational, open-license video (Big Buck Bunny – Blender Foundation).

Accessibility notes: Both iframes include descriptive title attributes, large readable dimensions, and consistent visual styling.
