Float‑Based Columns

Before Flexbox/Grid, floats positioned columns. Here, the left nav and right aside are floated; the footer clears them to avoid overlap.

/***** key bits *****/
.side-nav { float: left; width: 20%; }
.sidebar { float: right; width: 24%; }
.content { margin: 0 26% 0 22%; /* space for floats */ }
footer.site-footer { clear: both; }

Expandable Details

How does clear: both; work?

It forces the element to sit below any left/right floats, preventing overlap. We apply it to the footer.

Why not just use Flexbox/Grid?

Flexbox/Grid are modern and preferred, but floats are important for maintaining legacy layouts and understanding CSS history.
